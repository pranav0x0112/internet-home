Admonitions Examples

GitHub-style admonitions are supported! They provide a clean, semantic way to highlight important information in your content.

Multi-line Content

Admonitions support multiple paragraphs and other markdown formatting:

[!TIP] You can include bold text, italics, and code snippets in admonitions.

You can also have multiple paragraphs by continuing with the > marker on subsequent lines.

Styling

Each admonition type has a distinct color accent on the left border:

  • Note - Blue (#4da3ff)
  • Tip - Green (#3fb950)
  • Important - Purple (#a371f7)
  • Warning - Orange (#f0883e)
  • Caution - Red (#ff6b6b)

The styling is dark theme friendly and includes light theme support. Custom types use a neutral gray color.
